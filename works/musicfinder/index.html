Music Finder

Live site Code

A mobile-optimized webapp created as a project for a mobile programming course. The app allows you to find similar bands to the ones you like.

It uses Last.FM's API for band and album information and Geonames's API for local top-list discovery. The app uses mainly jQuery and AJAX for its functionality. The app has some responsivity but was not fully refined beyond mobile browser.
