The Conscript Committee

A set of posters created for a local branch of The Conscript Committee of The Finnish Defence Foreces.

The posters were made for various clubs and events managed by the Committee. The task for the sports clubs' posters was to create a consistent and appealing theme, with some aggressive undertones.

The posters were created mainly with Illustrator with some post work in Photoshop.
